# Add an Object

**Add an Object**

* Shift + A ⇒ Select type of Object
* Add bar (or F9) ⇒ Adjust dimensions / resolution / location **(only available at that moment)**
* Select other object ⇒ Dimensions  / resolution / location confirmed
