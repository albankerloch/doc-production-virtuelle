**Solidify Modifier**

Generate ⇒ Solidify : Add Tickness to an Object

Offset : set to 1 to add Thickness above (in external part)

Edge Data ⇒ Crease Inner : set to 1 to add sharp thickness in the inner part (but not in external part)
