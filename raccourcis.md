# Raccourcis

Object Mode

| Bouton                      | Action                            |
| --------------------------- | --------------------------------- |
| Scroll                      | Zoom in / out                     |
| Shift + A                   | Ajouter un nouvel objet           |
| Middle Mouse + Drag         | Orbiter (autour du point central) |
| Shift + Middle Mouse + Drag | Pan (autour du point central)     |
